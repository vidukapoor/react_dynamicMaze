# react.js dynamic_maze
---
first, you should install node and npm.

1. install npm packages 
    `npm install`

2. start server  
    `npm start`

3. open browser: [http://localhost:7777](http://localhost:7777)

## Insructions (Key usage)
1. w (move_up)
2. s (move_down)
3. a (move_left)
4. d (move_right)
